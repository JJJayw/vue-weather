# vue-weather

使用Vue、Vue-router、axios、tailwind css实现的天气预报APP

## 项目结构

* assets 静态资源
* components vue组件
  * **skeleton 骨架**
  * **weather 天气图标库**
* resource api密钥
* router 路由
* types 定义
* views 路由页面
  
## api

使用高德地图API查询行政区域划分，使用彩云天气API查询天气预报
在src/api文件夹中增加api.json
```
data.cityApiKey
data.weatherApiKey
```
